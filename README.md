# wiseflow-dev.github.io

Personal portfolio of **Mohamad Alhaj Ismail** — AI-native software builder.

Live at **[wiseflow-dev.github.io](https://wiseflow-dev.github.io/)**.

Plain HTML/CSS/JS, no frameworks, no build step. Self-hosted fonts, lazy-loaded media, light/dark theme, `prefers-reduced-motion` support.

## Structure

- `index.html` — the whole page
- `styles.css` — design system (warm paper + forest green, light & dark)
- `main.js` — theme toggle, scroll reveals, video play/pause on visibility
- `assets/` — fonts, WiseFlow screenshots, Motomation preview clips

## Develop

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server
```
